Interactive Fluid Simulation

Summary

We implement a fluid simulation that accounts for external forces such as user interaction. This allows for more realistic fluid simulation. We extend our work based on the paper.

Team Members 

Vikranth Srivatsa, Shunyuet (Benjamin) Cheung, Bhuvan Basireddy, Kevin Lo

Problem Description

        A lot of past projects in CS184 dealt with fluid simulations but do not account for the external forces such as user interaction in a realistic way. We try to use realistic fluid simulations that allow for user interactions and swirling the water. This can allow for simulations of water that have whirlpools and such.

Goals and Deliverables

        We plan on producing realistic simulations by modeling fluids with particles. The math behind the scenes would include formulas that consider pressure, surface tension, viscosity, and external forces. An example of what we hope to achieve is pouring water into a glass, which can be found in the paper listed in the “Resources” section of this proposal.

Here the water is being poured into a glass.

  Here the water is sloshing and settling.

Since our hope is that we want to create an interactive demo, we are trying to either render a website demo and use the mouse in order to display the interaction. In terms of measuring quality, we hope to measure the quality using a visual test of how the liquid simulation looks. It’s hard to measure quantitatively so we hope that the graphics demo is a good way to display the simulation.

If things go ahead of schedule, we hope to simulate other liquids and fluids. It would also be interesting to look into seeing when the container itself moves over the simulation time. It would also be interesting to see how ice is acting in the water.
